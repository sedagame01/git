Tipos de Posicionamiento CSS proporciona varias opciones de posicionamiento, cada una con su propio comportamiento y propósito se maneja con la propiedad position: 1. Posicionamiento Estático (static): Es el valor por defecto para todos los elementos. Lo usamos cuando no se necesita un control especial sobre la posición de un elemento. Los elementos se posicionan uno tras otro en el flujo del documento, lo que es ideal para layouts sencillos.

2. Posicionamiento Relativo (relative): Un elemento con position: relative se posiciona en relación con su posición original en el flujo del documento. Es decir, el elemento se mueve desde donde estaría normalmente, sin alterar el espacio que ocupa en el flujo. Permite ajustar la posición de un elemento ligeramente sin afectar el layout general. Desplazamiento: Puedes usar las propiedades top, right, bottom, y left para mover el elemento en relación con su posición original.

3. Posicionamiento Absoluto (absolute): Un elemento con position: absolute se elimina del flujo normal del documento y se posiciona en relación con su contenedor más cercano que tenga un posicionamiento distinto de static. Ubicación precisa: El elemento se puede colocar en cualquier lugar dentro de su contenedor, usando top, right, bottom, y left.

4. Posicionamiento Fijo (fixed): Útil para elementos que deben permanecer en la misma posición en la pantalla, incluso cuando se hace scroll. Comúnmente usado para menús de navegación, barras superiores, o botones de retorno al inicio.

5. Posicionamiento Adherido (sticky): Combina características de relative y fixed. El elemento se comporta como relativo hasta que el usuario hace scroll y alcanza una posición definida, donde se "adhiere" a la ventana gráfica. Ideal para menús que se desplazan con la página hasta un punto y luego permanecen fijos.
